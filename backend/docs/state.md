  - `POST /api/auth/reset-password` — redefine a senha após validação

- **Rate Limiting:** 5 req/min para solicitar, 10 req/min para validar, 5 req/min para redefinir

### Frontend
- **Páginas:**
  - `ForgotPasswordPage.tsx` — solicita email do usuário
